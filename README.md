# thedevbones crypt

My personal website hosted on [Neocities](https://thedevbones.neocities.org/). A place to share my games, development progress, and other fun stuff.

Built with no frameworks or build tools, just plain HTMl, CSS, and JavaScript. Made to look like the site is straight out of Web 1.0.

## Structure

```
public/
├── css/
│   └── style.css
├── about/
│   └── index.html
├── games/
│   └── index.html
├── gallery/
│   └── index.html
├── images/
├── buttons/
├── footer.html
├── index.html
└── robots.txt
```

## Deployment

Pushes to `main` are automatically deployed to Neocities via [deploy-to-neocities](https://github.com/marketplace/actions/deploy-to-neocities) GitHub Action.
